# Marie Curie: Key Achievements

**First Woman to Win a Nobel Prize** (1903 - Physics)
- Shared with Pierre Curie and Henri Becquerel
- For research on radiation phenomena

**First Person to Win Two Nobel Prizes** (1911 - Chemistry)
- For the discovery of radium and polonium
- For the isolation of radium

**Pioneering Scientific Contributions**:
- Developed the theory of radioactivity
- Techniques for isolating radioactive isotopes
- Discovery of two elements: **polonium** and **radium**

---

# Research Methodology

Curie's approach combined rigorous experimentation with innovative techniques:

**Sample Processing**
- Processed tons of pitchblende to isolate radium
- Developed fractional crystallization methods

**Measurement Techniques**
- Created the **Curie balance** for measuring radioactive emissions
- Developed precise calorimetric methods

# Key Formulas in Radiation Physics

**Radioactive Decay Law**:

```
N(t) = N₀ × e^(-λt)
```

**Half-life Relationship**:

```
t₁/₂ = ln(2) / λ ≈ 0.693 / λ
```

**Activity Calculation**:

```
A = λN = A₀ × e^(-λt)
```

Where:
- `λ` = decay constant
- `N` = number of radioactive atoms
- `t` = time
